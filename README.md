# Amaya PHP Framework

基于Composer完成的MVC框架,这是我开发PHP过程中最轻松的一次。

感谢###后盾网向军老师###，提供了PHP各种学习视频，让我从视觉设计、前端再一次跨进后端开发成为全栈设计师。。

此框架依赖第三方库：

Medoo 数据库

twig 模版引擎

whoops 调式错误

monolog 日志

macaw 路由

## 安装方式

不了解git下载的直接下载解压缩。

如果了解composer工具的，输入：

```html
composer require tokyo-lei/amaya
```
然后进行依赖下载（包括解压缩必须用composer工具执行）：
```html
composer update
```
